# Configs & Dotfiles

My personal configuration files for development tools and system customization on Windows.

## What's Inside

- `docker` - Docker configuration files (submodule)

- `dotfiles` - Configuration files for various tools
  - `code` - Editor settings and keybindings
  - `git` - Global git configuration
  - `powershell` - Profile and aliases
  - `powertoys` - PowerToys settings
  - `terminal` - Terminal appearance and behavior

> The `docker` directory contains semi-private Docker configurations intended for internal use. Access is restricted and available upon request.
