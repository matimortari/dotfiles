# Dotfiles 📁🔧

My personal configuration files, along with automation scripts for development tasks.

## 📂 Project Structure

This repository is organized into the following structure:

- **[/git/.gitconfig:](git/.gitconfig)** Git configuration.

- **[/docker/.config.json:](docker/.config.json)** Docker configuration.

- **[/powershell/Microsoft.PowerShell_profile.ps1:](powershell/Microsoft.PowerShell_profile.ps1)** PowerShell profile configuration.

- **[/code/:](code)** Visual Studio Code configuration folder.

  - **[settings.json:](code/settings.json)** Visual Studio Code settings.
  - **[keybindings.json:]()** Visual Studio Code keybindings.

- **[/scripts/:](scripts)** Automation scripts folder.

  - **[setup-nextjs.ps1:](scripts/setup-nextjs.ps1)** Quickstart a Next.js-powered website.
  - **[tree.ps1:](scripts/tree.ps1)** Custom implementation of the `tree` command for Windows which excludes large files.
